## Description du projet
**Gestion Bibliothèque** est une application web complète développée avec **Laravel** pour gérer les opérations d'une bibliothèque moderne.  
Elle permet aux administrateurs de gérer les livres, les catégories et les utilisateurs, et offre aux lecteurs une interface pour consulter et réserver des livres disponibles.  
L'objectif est de simplifier la gestion quotidienne et d'améliorer l'expérience utilisateur pour les lecteurs et le personnel de la bibliothèque.

---

## Fonctionnalités principales
- **Gestion des livres :** Ajout, modification, suppression, consultation.  
- **Gestion des catégories :** Organisation des livres par catégories pour un accès facile.  
- **Gestion des utilisateurs :** Création de comptes pour lecteurs et administrateurs.  
- **Gestion des réservations :** Les utilisateurs peuvent réserver un livre et suivre le statut de leur réservation.  
- **Interface administrateur :** Tableau de bord pour gérer les livres, les utilisateurs et les réservations.  
- **Interface utilisateur :** Consultation des livres, réservations et suivi des emprunts.  

---

## Technologies utilisées
- **Backend :** Laravel (PHP)  
- **Base de données :** MySQL  
- **Frontend :** Blade Templates, Bootstrap, CSS  
- **Versioning :** Git / GitHub
